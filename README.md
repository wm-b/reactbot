This is Discord application for ReactBot, capable of posting random or searchable reactions in any Discord chat and displaying information about Reaction usage.

This repository does not contain the raw reaction files, which are [available for purchase separately](https://jacksfilmscouncil-shop.fourthwall.com/en-gbp/products/reactbot-exe) from Jacksfilms.

ReactBot is a virtual reactor made to imitate low-effort reaction influencers, created by Jacksfilms and Camtoonist.
